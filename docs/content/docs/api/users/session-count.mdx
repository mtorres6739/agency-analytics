```
GET /api/sites/:site/users/session-count
```

Returns the number of sessions per day for a specific user. Useful for visualizing user engagement over time.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

<TypeTable
  type={{
    user_id: {
      description: 'User ID (device fingerprint or identified user ID)',
      type: 'string',
      required: true,
    },
    time_zone: {
      description: 'Timezone for date grouping',
      type: 'string',
      default: 'UTC',
      typeDescription: 'IANA timezone (e.g., "America/New_York")',
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of daily session counts',
      type: 'DailyCount[]',
    },
  }}
/>

#### DailyCount Object

<TypeTable
  type={{
    date: {
      description: 'Date',
      type: 'string',
      typeDescription: 'YYYY-MM-DD format',
    },
    sessions: {
      description: 'Number of sessions on this date',
      type: 'number',
    },
  }}
/>
