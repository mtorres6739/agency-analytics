```
GET /api/sites/:site/users/:userId
```

Returns detailed information about a specific user, including their profile traits and linked devices.

### Path Parameters

<TypeTable
  type={{
    userId: {
      description: 'User ID (device fingerprint or identified user ID)',
      type: 'string',
      required: true,
    },
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Response

<TypeTable
