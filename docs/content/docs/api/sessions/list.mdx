```
GET /api/sites/:site/sessions
```

Returns a paginated list of sessions with their analytics data.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      default: 1,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 100,
    },
    user_id: {
      description: 'Filter sessions by user ID',
      type: 'string',
      required: false,
    },
    identified_only: {
      description: 'Only return sessions from identified users',
      type: 'string',
      typeDescription: '"true" | "false"',
      default: 'false',
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of session objects',
      type: 'Session[]',
    },
  }}
/>
