```
GET /api/sites/:site/error-events
```

Returns individual error occurrences for a specific error message, with full context including stack traces.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    errorMessage: {
      description: 'The error message to filter by',
      type: 'string',
      required: true,
    },
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      required: false,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 20,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of error event objects',
      type: 'ErrorEvent[]',
    },
  }}
/>
