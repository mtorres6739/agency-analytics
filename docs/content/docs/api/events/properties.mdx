```
GET /api/sites/:site/events/properties
```

Returns property key-value pairs for a specific event name with their occurrence counts.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    event_name: {
      description: 'Event name to get properties for',
      type: 'string',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of property objects',
      type: 'EventProperty[]',
    },
  }}
/>

#### EventProperty Object

<TypeTable
  type={{
    propertyKey: {
      description: 'Property key name',
      type: 'string',
    },
    propertyValue: {
      description: 'Property value',
      type: 'string',
    },
    count: {
      description: 'Number of occurrences',
      type: 'number',
    },
  }}
/>
