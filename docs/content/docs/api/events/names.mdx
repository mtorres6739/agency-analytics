```
GET /api/sites/:site/events/names
```

Returns a list of unique custom event names with their occurrence counts.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters).

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of event name objects',
      type: 'EventName[]',
    },
  }}
/>

#### EventName Object

<TypeTable
  type={{
    eventName: {
      description: 'Name of the custom event',
      type: 'string',
    },
    count: {
      description: 'Number of occurrences',
      type: 'number',
    },
  }}
/>
