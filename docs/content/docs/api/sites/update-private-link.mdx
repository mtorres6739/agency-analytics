```
POST /api/sites/:siteId/private-link-config
```

Generates or revokes a private link key for sharing analytics without authentication. Requires admin or owner role.

### Path Parameters

<TypeTable
  type={{
    siteId: {
      description: 'Site ID',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Request Body

<TypeTable
  type={{
    action: {
      description: 'Action to perform',
      type: 'string',
      typeDescription: '"generate_private_link_key" | "revoke_private_link_key"',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    success: {
      description: 'Whether the request was successful',
      type: 'boolean',
    },
    data: {
      description: 'Updated private link configuration',
      type: 'object',
    },
  }}
/>

#### Data Object

<TypeTable
  type={{
    privateLinkKey: {
      description: 'New private link key (for generate), or null (for revoke)',
      type: 'string | null',
    },
  }}
/>
