```
GET /api/sites/:site/performance/overview
```

Returns aggregate performance metrics with percentile breakdowns for all Core Web Vitals.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) (time and filters).

### Response

<TypeTable
  type={{
    data: {
      description: 'Performance metrics object',
      type: 'PerformanceMetrics',
    },
  }}
/>

#### PerformanceMetrics Object

Each metric (lcp, cls, inp, fcp, ttfb) includes the following percentiles:

<TypeTable
  type={{
    lcp_p50: {
      description: 'LCP 50th percentile (median)',
      type: 'number',
      typeDescription: 'milliseconds',
    },
    lcp_p75: {
      description: 'LCP 75th percentile',
      type: 'number',
      typeDescription: 'milliseconds',
    },
    lcp_p90: {
      description: 'LCP 90th percentile',
      type: 'number',
      typeDescription: 'milliseconds',
    },
    lcp_p99: {
      description: 'LCP 99th percentile',
      type: 'number',
      typeDescription: 'milliseconds',
    },
    total_performance_events: {
      description: 'Total number of performance measurements',
      type: 'number',
    },
  }}
/>
