```
GET /api/sites/:site/performance/time-series
```

Returns performance metrics over time, broken down by configurable time buckets. Useful for charting performance trends.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "hour" | "day" | "week" | "month"',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Array of time-bucketed performance data',
      type: 'PerformancePoint[]',
    },
  }}
/>

#### PerformancePoint Object

Each point includes all percentiles for each metric plus:

<TypeTable
  type={{
    time: {
      description: 'Start of the time bucket',
      type: 'string',
      typeDescription: 'ISO 8601 datetime',
    },
    event_count: {
      description: 'Number of performance events in this bucket',
      type: 'number',
    },
  }}
/>
