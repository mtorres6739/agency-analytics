```
GET /api/sites/:site/performance/by-dimension
```

Returns performance metrics broken down by a specific dimension (page, country, browser, etc.). Useful for identifying performance issues on specific pages or for specific user segments.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    dimension: {
      description: 'Dimension to break down by',
      type: 'string',
      typeDescription: '"pathname" | "country" | "device_type" | "browser" | "operating_system" | "region"',
      required: true,
    },
    page: {
      description: 'Page number (1-indexed)',
      type: 'number',
      required: false,
    },
    limit: {
      description: 'Number of results per page',
      type: 'number',
      default: 100,
    },
    sort_by: {
      description: 'Field to sort by',
      type: 'string',
      default: 'event_count',
    },
    sort_order: {
      description: 'Sort direction',
      type: 'string',
      typeDescription: '"asc" | "desc"',
      default: 'desc',
    },
  }}
/>

### Response

<TypeTable
  type={{
    data: {
      description: 'Paginated response object',
      type: 'PaginatedResponse',
    },
  }}
/>

#### DimensionPerformance Object

Each object includes the dimension value plus all percentiles:

<TypeTable
  type={{
    pathname: {
      description: 'The dimension value (field name matches dimension param)',
      type: 'string',
    },
    event_count: {
      description: 'Number of performance events',
      type: 'number',
    },
    lcp_avg: {
      description: 'Average LCP',
      type: 'number',
    },
    lcp_p50: {
      description: 'LCP 50th percentile',
      type: 'number',
    },
    lcp_p75: {
      description: 'LCP 75th percentile',
      type: 'number',
    },
    lcp_p90: {
      description: 'LCP 90th percentile',
      type: 'number',
    },
    lcp_p99: {
      description: 'LCP 99th percentile',
      type: 'number',
    },
  }}
/>
