```
GET /api/organizations/:organizationId/members
```

Returns all members of an organization with their user details and roles. Requires membership in the organization.

### Path Parameters

<TypeTable
  type={{
    organizationId: {
      description: 'Organization ID',
      type: 'string',
      required: true,
    },
  }}
/>

### Response

<TypeTable
  type={{
    success: {
      description: 'Whether the request was successful',
      type: 'boolean',
    },
    data: {
      description: 'Array of organization members',
      type: 'array',
    },
  }}
/>

#### Member Object

Each member in the `data` array contains:

<TypeTable

      description: 'User ID',

    },

      typeDescription: 'ISO 8601 datetime',
    },
    user: {
      description: 'User details',
      type: 'object',
    },
  }}
