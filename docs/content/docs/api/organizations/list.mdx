```
GET /api/organizations
```

Returns all organizations the authenticated user is a member of, including all members for each organization. Excludes subscription and billing information. Supports both session cookies and API key (Bearer token) authentication.

<Callout type="warn">The `members` roster is only populated for session-cookie (dashboard) requests. API key / Bearer callers receive `members: []` for every organization — use [GET /organizations/:id/members](/docs/api/organizations/members) to retrieve member data with a key.</Callout>

### Response

Returns an array of organization objects.
