```
GET /api/sites/:site/live-user-count
```

Returns the count of unique active sessions within the specified time window. Useful for displaying real-time visitor counts.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

<TypeTable
  type={{
    minutes: {
      description: 'Time window in minutes to count active sessions',
      type: 'number',
      default: 5,
      typeDescription: 'Sessions with activity within the last N minutes are considered "live"',
    },
  }}
/>

### Response

<TypeTable
  type={{
    count: {
      description: 'Number of unique active sessions',
      type: 'number',
    },
  }}
/>

#### Time Window Guide

| Minutes | Use Case |
|---------|----------|
| `1` | Very active visitors (currently browsing) |
| `5` | Standard "live" count (default) |
| `15` | Recently active visitors |
| `30` | Short-term engagement window |

### Polling Example

```javascript title="Real-time Widget"
async function updateLiveCount() {
  const response = await fetch(
    'https://app.rybbit.io/api/sites/123/live-user-count?minutes=5',
    {
      headers: {
        'Authorization': 'Bearer your_api_key'
      }
    }
  );
  const { count } = await response.json();
  document.getElementById('live-visitors')
    .textContent = count;
}

// Update every 15 seconds
setInterval(updateLiveCount, 15000);
```
