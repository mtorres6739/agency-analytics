```
GET /api/sites/:site/overview/time-series
```

Returns time-series analytics data broken down by configurable time buckets. Useful for charting trends over time.

### Path Parameters

<TypeTable
  type={{
    site: {
      description: 'Site ID or site identifier',
      type: 'string | number',
      required: true,
    },
  }}
/>

### Query Parameters

Accepts all [Common Parameters](/docs/api/getting-started#common-parameters) plus the following:

<TypeTable
  type={{
    bucket: {
      description: 'Time bucket size for grouping data',
      type: 'TimeBucket',
      typeDescription: '"minute" | "five_minutes" | "ten_minutes" | "fifteen_minutes" | "hour" | "day" | "week" | "month" | "year"',
      required: false,
      default: 'hour',
    },
  }}
/>

#### Bucket Selection Guide

| Bucket | Best For |
|--------|----------|
| `minute` | Real-time monitoring (last hour) |
| `five_minutes` | Short-term analysis (last few hours) |
| `hour` | Daily analysis |
| `day` | Weekly/monthly analysis |
| `week` | Monthly/quarterly analysis |
| `month` | Yearly analysis |

### Response

<TypeTable
  type={{
    time: {
      description: 'Start of the time bucket',
      type: 'string',
      typeDescription: 'Datetime string, "YYYY-MM-DD HH:MM:SS"',
    },
    sessions: {
      description: 'Number of sessions in this bucket',
      type: 'number',
    },
    pageviews: {
      description: 'Number of pageviews in this bucket',
      type: 'number',
    },
    users: {
      description: 'Unique users in this bucket',
      type: 'number',
    },
    pages_per_session: {
      description: 'Average pages viewed per session',
      type: 'number',
    },
    bounce_rate: {
      description: 'Percentage of single-page sessions',
      type: 'number',
      typeDescription: 'Range: 0-100',
    },
    session_duration: {
      description: 'Average session duration in seconds',
      type: 'number',
    },
  }}
/>
